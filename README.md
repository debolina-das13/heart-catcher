# Basket Heart Game 

A simple browser-based game where users move a basket to catch falling hearts.  
This game helps improve **hand-eye coordination, reflexes, and focus** while providing a **fun and engaging experience**.

---

## Features
- Full-screen pink & white gradient background
- Movable basket using **keyboard arrows** or **mouse**
- Falling hearts animation
- Score tracking
- Simple and beginner-friendly code

---

## Controls
- **Mouse movement:** Move the basket left and right  
- **Keyboard:** Use **Arrow Left** and **Arrow Right** keys  

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM manipulation & events)
